# REMA — Premium Artist Landing Page

A high-end, cinematic landing page concept for Nigerian Afro-Rave pioneer **Rema** (Divine Joshua Ikubor). This is a record label outreach concept showcasing award-winning web design principles.

## 🎯 Design Philosophy

**Dark Luxury Aesthetic** — Inspired by premium music artist websites and Awwwards-level design:
- Deep blacks, charcoal tones, subtle red accents
- Clean, minimal, bold typography
- Strong visual hierarchy
- Intentional animations and transitions
- No generic AI patterns

## ✨ Features

### Hero Section
- Fullscreen cinematic video background with gradient overlay
- Floating particle system
- Smooth parallax scrolling
- Animated text reveals
- Live stats strip (2B+ streams, Billboard #3, etc.)

### Sections
1. **About** — Split layout with image + biography
2. **Music Highlights** — Interactive cards with hover effects (Calm Down, Rave & Roses, HEIS)
3. **Visual Gallery** — Modern grid with lightbox functionality
4. **Impact** — Animated statistics and achievements
5. **Press** — Featured publications
6. **Contact** — Professional inquiry form for record labels

### Animations
- GSAP-quality CSS animations
- Scroll-triggered reveals (fade + slide)
- Parallax effects on hero
- Image hover transformations (scale, blur, overlay)
- Counter animations for stats
- Smooth section transitions
- 3D tilt effect on music cards
- Cursor glow effect

### Technical
- **Pure HTML, CSS, JavaScript** — No frameworks
- Fully responsive (desktop, tablet, mobile)
- Optimized performance
- Smooth 60fps animations
- Intersection Observer API for scroll reveals
- Mobile-friendly navigation

## 🚀 Usage

Simply open `index.html` in a modern browser. No build process required.

```bash
# Option 1: Direct open
open index.html

# Option 2: Local server (recommended)
python -m http.server 8000
# or
npx serve
```

Then visit `http://localhost:8000`

## 🎨 Design System

### Colors
- **Black**: `#080808` — Primary background
- **Charcoal**: `#111111`, `#161616`, `#222222` — Layered surfaces
- **Red**: `#c0392b` — Brand accent
- **White**: `#f5f5f0` — Primary text
- **Gold**: `#c9a84c` — Special highlights

### Typography
- **Display**: Bebas Neue — Headlines, large text
- **Body**: Inter — Paragraphs, UI elements
- **Serif**: Playfair Display — Italics, quotes

### Spacing
- Fluid responsive spacing using `clamp()`
- Consistent grid systems
- Perfect alignment throughout
